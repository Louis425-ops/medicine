# 高血压共病辅助用药决策系统

基于 `schedule.md` 设计文档实现的课程原型，聚焦高血压合并常见共病场景下的辅助用药推荐、风险排除与可解释展示。

## 当前版本能力

- 首诊录入页：覆盖基础信息、高血压主病信息、7 类共病模块、当前用药与既往风险
- 规则引擎：内置首版药物知识库与规则库，支持排除、慎用、加权和监测提示
- 推荐结果页：展示 Top 3 药物类别、代表药物、排除药物、慎用药物、命中规则和监测建议
- 解释图谱页：三层结构展示“患者特征 -> 规则 -> 药物”关系
- 病例管理：支持示例病例加载和本地病例保存回放

## 运行方式

```bash
npm install
npm run dev
```

生产构建：
